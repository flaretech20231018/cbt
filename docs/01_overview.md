# プロジェクト概要

## プロジェクト名

**CBT 思考記録アプリ**（仮称）

> 正式名称は UI / ブランディング検討フェーズで決定する

---

## ビジョン

> CBT（認知行動療法）を学びたいすべての人が、いつでもどこでも3コラム法を実践できる場を提供する

---

## 解決する問題

### 1. カウンセリングへのアクセス困難

- カウンセリングはコスト・時間・心理的ハードルが高く、多くの人が継続的なサポートを受けられない
- 「CBTを試してみたい」という軽度な動機を持つ人が、入口に立てていない

### 2. 記録の継続・振り返りが難しい

- 紙とペンでは「どこに書いたか」「過去の記録が見当たらない」問題が起きやすい
- 時系列で記録を管理・振り返る手段がない

---

## ターゲットユーザー

| 属性 | 内容 |
|------|------|
| 主要ユーザー | CBTを学びたい一般ユーザー |
| 前提知識 | CBT初心者〜入門者（専門知識は不要） |
| 利用場面 | ストレスを感じた時、気分が落ち込んだ時、日々のセルフケアとして |

> 詳細なペルソナは [02_personas.md](./planning/02_personas.md) を参照

---

## MVP スコープ

### In Scope（初期リリースで実装する）

- ユーザー認証（サインアップ / ログイン / ログアウト）
- 3コラム法の思考記録作成・閲覧・編集・削除（CRUD）
- 記録の時系列一覧表示
- レスポンシブ対応（PC / スマートフォン）

### Out of Scope（MVP 以降で検討）

- AIによるバランス思考サジェスト機能
- 継続を促す仕組み（ストリーク、リマインダー、進捗可視化）
- ソーシャル機能・共有機能
- 多言語対応（i18n）
- 認知の歪みタイプの自動分類

---

## 成功指標（KPI）

| 指標 | 目標 |
|------|------|
| 操作完了時間 | 初回でも3分以内に1記録を作成できる |
| 継続利用 | 週1回以上、1ヶ月継続して記録できるUXを提供する |
| 学習支援 | CBT初心者が3コラム法の意味を理解しながら記録できる |

---

## 将来的な展望

1. **継続促進機能**: ストリーク表示・リマインダー・記録の振り返りガイド
2. **AI連携**: Claude API を利用したバランス思考のサジェスト
3. **インサイト**: 気分・思考パターンの可視化（グラフ・タグ分析）
4. **認知の歪み学習**: 代表的な認知の歪みパターンの解説・ラベリング機能
