# ペルソナ定義

> 本アプリのターゲットは「ストレスや生活の閉そく感を抱えている一般ユーザー」。
> 「CBTを学びたい」という明確な動機を持つとは限らず、
> 「なんかしんどい」「自分が悪いのかも」という曖昧な状態のままアプリに辿り着く人を含む。
> 精神科・心療内科への受診経験はなく、自分の状態をメンタル不調と認識していないケースが多い。

---

## ペルソナ 1（メイン）: 山田 圭佑

| 項目 | 内容 |
|------|------|
| 年齢 | 35歳 |
| 職業 | IT企業 勤務（エンジニア） |
| 居住 | 東京都内、一人暮らし |
| デバイス | スマートフォン（メイン）、PC（業務） |

### 背景・状況

- 仕事のプレッシャーや上司との関係で、最近なんとなくしんどい日が続いている
- 自分の状態を「ストレス」「メンタル不調」とは認識しておらず、**「仕事が合っていないだけ」「自分が弱いだけ」「もっと頑張れるはず」**と解釈している
- 気分の落ち込みや閉そく感はあるが、それが思考パターンから来ているとは気づいていない
- 自己啓発本やビジネス書は読む。仕事後にスマホで記事を眺める習慣がある
- 「ストレス 対処法」「気分 切り替え」などで検索することがあり、そこでCBTに初めて触れる可能性がある

### ゴール（本人が意識しているゴール）

- 特に明確なゴールはない。「なんかこのしんどさをどうにかしたい」という漠然とした感覚
- 自己改善・仕事の効率化への関心から、ツールやアプリを試すことはある

### ゴール（アプリが提供したい価値）

- 気持ちを言語化することで「自分はこんなことを考えていたのか」という気づきを得る
- 繰り返す思考パターンに少しずつ気づき、自分を客観視できるようになる

### ペインポイント

- 「気分が悪い」「しんどい」という感覚はあるが、原因を言語化できていない
- 「病院に行くほどじゃない」と思っているため、専門的な助けを求める選択肢が頭にない
- 自分の責任・努力不足として処理しているため、思考パターン自体を疑う視点がない
- 振り返りや内省の習慣がなく、同じ状況で同じ感情を繰り返していることに気づいていない

### 行動特性

- 空き時間（通勤電車・昼休み）にスマホでアプリを触る
- 「試しに使ってみる」ハードルは低いが、価値を感じないとすぐ離脱する
- アプリに「心理・メンタル」色が強すぎると抵抗を感じる可能性がある

### このアプリへの期待

> 「なんとなく使ってみたら、いつも頭の中でぐるぐるしていたことをうまく書き出せた。
> 読み返したら『あ、自分はこういう時にこう考えるんだ』ってわかった気がした。
> そういうシンプルな気づきがあれば、続けてみようかなと思える。」

## ペルソナ 2（サブ）: 佐藤 さくら

| 項目 | 内容 |
|------|------|
| 年齢 | 24歳 |
| 職業 | 大学院生（文系） |
| 居住 | 大阪市内、一人暮らし |
| デバイス | スマートフォン（メイン） |

### 背景・状況

- 修士論文・就活・将来への不安が重なり、漠然とした焦りを感じている
- SNS・YouTube で心理学・メンタルヘルス系のコンテンツをよく見る
- 「CBT」や「認知の歪み」という言葉は知っており、「試してみたい」と思っていた
- HSP という概念に共感している。感情の波が激しく、他者の言葉に傷つきやすい

### ゴール

- 自分の「考え方の癖（認知の歪み）」を理解して、少しずつ変えたい
- 感情に流されず、物事をもう少し客観的に見られるようになりたい

### ペインポイント

- 「3コラム法を試そう」と思っても、何を書けばいいか迷って手が止まる
- 紙のノートに書いたが、あとから読み返すのが恥ずかしくて続かなかった
- アプリを使い始めても、操作が複雑だったり手順が多いと途中で辞めてしまう

### 行動特性

- 新しいアプリを気軽にインストールして試す
- UIが整っていないと使い続けない
- 「他の人も同じことで悩んでいる」と知ると安心する

### このアプリへの期待

> 「各コラムに何を書けばいいかのヒントが欲しい。
> 記録するだけで、なんとなく頭の中が整理される感覚があると嬉しい。」

## ペルソナから見えてくる設計方針

| 観点 | 方針 |
|------|------|
| アプリの入口 | 「CBTアプリ」ではなく「気持ちを整理するアプリ」として提示。心理的ハードルを下げる |
| UI言語 | 「認知の歪み」「自動思考」などの専門用語を前面に出さない。補足説明を添える |
| 初回ユーザー体験 | 「何を書けばいいか」を迷わせない（各コラムに具体的なプレースホルダー・ヒントを表示） |
| 継続性 | 記録一覧で過去を振り返れる時系列表示（「あの時と同じパターン」への気づきを促す） |
| 軽量さ | 入力ステップを最小化し、「さっと書いてさっと終わる」体験 |
| 医療的免責 | 「このアプリは医療行為ではありません」を適切な場所に明示 |
